# MakerWorld 油猴脚本使用说明

## 目录
- 脚本文件：`plugin/tampermonkey/mw_quick_archive.user.js`

## 安装
1. 浏览器安装 Tampermonkey
2. 新建脚本，粘贴 `mw_quick_archive.user.js` 内容并保存
3. 或直接拖拽该文件到 Tampermonkey 导入

## 功能
- 在模型页生效：
  - `https://makerworld.com.cn/zh/models/*`
  - `https://makerworld.com/zh/models/*`
- 菜单配置后端 API 地址
- 手动设置并同步 Cookie
- 菜单重新下载缺失 3MF 文件
- 一键归档当前模型（右下角浮动按钮 + 菜单命令）
- 归档开始和完成时都会弹出明显提示

## 菜单项
- `⚙️ 设置后端地址与手动 Cookie`：配置本地后端地址，保存并同步手动 Cookie
- `归档当前模型`：对当前模型页发起归档
- `重新下载缺失 3MF 文件`：调用后端重试 `missing_3mf.log` 中记录的缺失实例
- `🍪 手动 Cookie 同步到后端`：将当前已保存的手动 Cookie 再次同步到后端

## 使用流程
1. 打开任意 MakerWorld 模型页
2. Tampermonkey 菜单点击「⚙️ 设置后端地址与手动 Cookie」
3. 填写 API 地址、手动 Cookie，点击保存并同步
4. 点击页面右下角「📦 归档模型」即可归档
5. 如遇缺失 3MF，可在 Tampermonkey 菜单点击「重新下载缺失 3MF 文件」触发补下载
