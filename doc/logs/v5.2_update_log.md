# v5.2 更新日志

> 更新日期: 2026-03-04

## 更新概述
本次更新新增全站主题体验升级：支持亮色/暗黑模式切换，并将在线详情页与主页主题联动，保证浏览体验一致。

## 详细变更

### 新增功能
- 新增主题切换能力（亮色 / 暗黑）：
  - 主页（模型库）支持一键切换
  - 配置页支持一键切换
- 在线详情页（`/v2/files/{model_dir}`）新增主题同步：
  - 自动跟随主页当前主题
  - 无需单独配置

### 优化改进
- 主题状态持久化到浏览器本地存储，刷新页面后保持用户选择。
- 页面首屏在 `<head>` 提前注入主题，减少切换闪烁感。
- 主页、配置页、在线详情页补充暗黑模式变量适配，卡片、面板、悬停态在暗黑下更统一。

### 保持不变
- 本地离线页打印按钮策略不变：仅在线版本显示打印按钮。

## 涉及文件
- `app/static/js/theme.js` — 主题切换与主题同步逻辑
- `app/static/css/variables.css` — 主题变量与暗黑模式覆盖
- `app/static/css/gallery.css` — 模型库页主题适配
- `app/static/css/config.css` — 配置页主题适配
- `app/static/css/model.css` — 在线详情页主题适配
- `app/templates/gallery.html` — 主题按钮与脚本接入
- `app/templates/config.html` — 主题按钮与版本号更新
- `app/templates/model.html` — 在线详情页主题跟随接入
- `README.md` — 当前版本与说明更新
