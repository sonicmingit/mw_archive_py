# v5.5.1 更新日志

> 更新日期: 2026-03-10

## 更新概述
本次版本聚焦“在线模型可维护”和“首页模型库管理”两条主线。在线模型现在支持直接在详情页编辑归档内容；首页则新增收藏夹能力与简约显示模式，方便对大量模型进行整理、筛选和快速浏览。

## 当前版本同步说明
- 当前版本：`v5.5.1`
- 更新说明：[`doc/logs/v5.5.1_update_log.md`](./v5.5.1_update_log.md)
- 本次重点：
  - 在线模型支持直接编辑归档内容；
  - 首页新增收藏夹功能；
  - 首页支持简约显示模式。

## 详细变更

### 新增功能
- 新增在线模型详情页编辑能力：
  - 支持直接修改标题、标签、分类、版本备注与简介
  - 支持设计图管理、封面调整与保存前自动备份
  - 支持恢复最近一次备份，降低误操作风险
- 新增首页收藏夹能力：
  - 支持创建收藏夹并填写名称、简介
  - 支持多选模型后批量加入已有或新建收藏夹
  - 支持按收藏夹筛选模型
- 新增首页简约显示模式：
  - 卡片改为高密度图片浏览模式
  - 使用悬浮标题方式展示模型名称

### 优化改进
- 首页多选模式增强：
  - 增加多选按钮高亮反馈
  - 批量操作条集中展示“加入收藏夹 / 批量删除 / 取消选择”
- 首页收藏夹交互优化：
  - 收藏夹入口调整到左侧分类标签上方
  - 在当前收藏夹视图下支持将已选模型移出该收藏夹
- 简约显示模式交互优化：
  - 模型名称以半透明方式悬浮在图片下方
  - 鼠标悬浮时可滚动显示较长标题

### Bug 修复
- 修复首页多选与收藏夹交互中状态反馈不够明确的问题
- 修复简约模式下标题信息展示不清晰的问题

## 兼容性与注意事项
- 本次不涉及破坏性数据迁移，旧版 `gallery_flags.json` 会自动兼容扩展后的收藏夹结构
- 简约显示模式当前按浏览器本地 `localStorage` 记忆，不做服务端全局共享
- 收藏夹当前支持创建、加入、筛选、从当前收藏夹移除模型，暂未提供重命名和删除收藏夹入口
